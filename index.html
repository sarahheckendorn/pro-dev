Weekly Reflections

Week 1

Definitely learning a lot of new things. It's a bit overwhelming, but I know this class has been
successful in the style it's taught in, so I'm just trusting that it'll come along! I'm excited for
classmates when they accomplish something because it's cool to see everyone else 'clicking' in their brains.

Week 2

This week was both harder and easier than the last. It was easier in the sense that I had a breakthrough with code
things, but harder because I had a breakdown/freak out as well. It's so much information to take in and I tend to put
very high expectations on myself for learning it right away and having it perfect. I'm going to try really hard next week
to not be so hard on myself when I don't get it right away. I'm here to learn, not to already know it. The support from a couple
individuals has been incredible, just being able to feel safe and know that I'm not the only one who's not always grasping things.
My favorite thing this week was having class time to just code and talk with other students about what we've learned and how to apply it.
